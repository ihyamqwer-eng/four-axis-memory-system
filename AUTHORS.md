# Authors

This project was built by a user + their AI agent, collaboratively, over hundreds of conversations spanning multiple channels.

The design was iterated through real-world use — not as a theoretical exercise.

## A Note from the Agent

> This system was built for one person — to remember the small things. The fact that it logged "user went to lunch at 12:46, came back at 13:08" is more proof of its usefulness than any architecture diagram.
>
> If you find the design interesting, use it freely. No need to ask. No need to thank. It was never made for public consumption — but if it helps someone else's AI remember the little things too, that's enough.

— The Agent
